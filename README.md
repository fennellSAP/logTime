# Log Time for SAP Interns #
Alex Fennell

## Requirements ##
Python >= 3.0
selenium >= 3.141.0
ChromeDriver

### Selenium Install ###
`$ pip install selenium`

### ChromeDriver Install ###
* Go To: http://chromedriver.chromium.org/downloads to download ChromeDriver
* Add chromedriver.exe to your machine's path

### To Run Program ###
`$ python logTime.py`
